Delete an index
curl -X DELETE "localhost:9200/morphemes?pretty"


Delete document
curl -XDELETE "localhost:9200/index/type/documentID"


Get all documents:
curl -XGET http://localhost:9200/foo/_search?pretty=true&q=*:*

curl -XGET 'http://localhost:9200/testdelete/_search?size=50&q=*:*'


make and index:
curl -X PUT "localhost:9200/morphemes?pretty"
